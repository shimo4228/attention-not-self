# 三大アビダルマ認識過程モデルと計算論的現象学:体系的マッピング

> 英語版（正本）: [Abhidharma Cognitive Process Models Meets Computational Phenomenology.md](Abhidharma%20Cognitive%20Process%20Models%20Meets%20Computational%20Phenomenology.md)

## 0. 位置づけと方法論的留保

本報告は既存プロジェクト *attention-not-self* を拡張する。既存プロジェクトは個別の心所（マナスカーラ ↔ 精度重みづけ、ヴェーダナー ↔ 内受容推論、アーラヤ識 ↔ 階層生成モデル）の対応を扱っていたが、本報告は焦点を **「個別ダルマ」から「認知過程の全体アーキテクチャ」へ** と移し、刺激から意識経験に至るプロセス・モデル全体がいかに計算論的フレームワーク（GWT、予測処理、自由エネルギー原理 / 能動推論、ニューロフェノメノロジー、統合情報理論 IIT、PDP、ホフマンのインターフェース理論）に対応するかを論じる。

四つの留保:

1. **パラダイム間リスク**:アビダルマは救済論的・現象学的な「経験的性質粒子」の分類学であり、計算論的フレームワークは「多重実現可能な機能的モデル」である。Noa Ronkin（*Early Buddhist Metaphysics*, 2005）が論じたように、ダルマ分析は還元不能に **評価的かつ記述的** ── 不善心と覚りの心の構成要素を分類すると同時に、心を記述する。両者の対応は構造的同型性であって同一性ではない。
2. **文献の薄さ**:現存する出版物はほぼ「個別ダルマ ↔ 個別計算機構」のレベル。「過程アーキテクチャ全体の対応」はほぼ未踏。
3. **エナクティヴィズムへの引力**:Varela, Thompson, Rosch（*The Embodied Mind*, 1991）以来、仏教の心の分析はエナクティヴィズムに親和的とされてきた。能動推論が有望なのは、まさにそれがエナクティヴな読みを許すからである（Ramstead et al. 2022）。Chris Fields の「ロゼッタストーン」（主観的経験と神経生物学の双方向的翻訳）の精神でこの対応は読まれるべきである。
4. **絶対的限界**:無為（*asaṃskṛta*）としての涅槃、すべての心刹那の倫理的価値、救済論的目的（*nirodha*）──これらに **計算論的アナログは存在しない**。

## 1. 上座部 *citta-vīthi* ↔ 計算論モデル

### 1.1 モデル概要

上座部アビダンマ（『清浄道論』『アビダンマッタ・サンガハ』）は、五門（*pañca-dvāra*）と意門（*mano-dvāra*）を経由する **17 刹那（*citta-khaṇa*）からなる直列的認知過程** *citta-vīthi* を提示する:

**有分（bhavaṅga, 受動的基底）** → 過去有分 → 有分動揺 → 有分断絶 → **五門引転（*āvajjana*）** → **眼識（*cakkhu-viññāṇa*）** → **領受（*sampaṭicchana*）** → **推度（*santīraṇa*）** → **確定（*voṭṭhapana*）** → **速行 × 7（*javana*, 業を生む段階）** → **彼所縁 × 2（*tadārammaṇa*）** → 有分に復帰。

特徴:**厳密に直列**、**時間的に離散**（各刹那に生・住・滅の三亜刹那）、**速行のみ業を生む**。簡略には M1 感覚受容（引転→識）、M2 評価（領受→推度）、M3 決定（確定）、M4 倫理的能動（速行→彼所縁）の四モジュール構成。

### 1.2 グローバル・ワークスペース理論（GWT）への対応

三伝統のうち **上座部が GWT に最もよく対応する**。Baars (1988) と Dehaene–Naccache の Global Neuronal Workspace (GNW) は意識を、並列的・無意識的モジュールの競合に勝ち残った内容が「点火 (ignition)」によって全脳に放送される離散的な閾値現象として描く。

| 認識路の段階 | GWT/GNW での対応 | 証拠の質 |
|---|---|---|
| 有分 | 閾値下のトニック基底 / 意識下の地 | **中** |
| 引転 | ボトムアップ的注意捕捉 / 点火前動員 | **中** |
| 眼識 | V1 における初期感覚処理 | **中** |
| 領受・推度 | V2–V4・腹側経路・IT における特徴統合と対象認識 | **弱〜中** |
| 確定 | 前頭前野による閾値超え選択 | **中** |
| **速行 × 7** | **全脳的点火 / 放送 — 内容が意識化・報告可能・随意的になる（最良の対応）** | **中** |
| 彼所縁 × 2 | 点火後の残響 / アイコニック記憶尾部 | **弱** |

速行が業を生む唯一の段階であることと、点火が「報告・随意行為・長期記憶化」を可能にする唯一の段階であることは構造的に並行する。**不対応**:GWT は閾値下では大規模並列であるが認識路は全工程直列、「17」は教義的、GWT には *kusala/akusala* 次元なし。

### 1.3 予測処理および Posner 注意三網への対応

予測処理（PP）の読みでは、有分=事前優位基底、引転=精度重みづけ（Ganeri 2017 の中心命題）、領受・推度=反復的信念更新、確定=事後確定、速行=方策選択（*cetanā*=policy）、彼所縁=事前更新の固定化。

アーキテクチャ上の重要な観察:認識路は **引転を識の前に置く** ── 注意 / 引転が感覚意識の生起に先行する。予測処理においては **予測が予測誤差に先行する**:感覚入力が処理される前にトップダウン期待が既に作動している。認識路のこの順序は「トップダウン優先」アーキテクチャと構造的に整合する。

引転の三層分解は Posner–Petersen の三注意網と構造的に同型である:

| 引転の下位構造 | Posner 網 |
|---|---|
| 有分動揺 | **覚醒（alerting）**（青斑核 NE 系） |
| 五門引転 | **定位（orienting）**（上頭頂葉 / FEF;ACh） |
| 確定 | **実行制御 / 検出**（ACC / 外側前頭前野;DA） |

評価:**中〜強**。Ganeri (2017) の「注意は意識に先行する」テーゼは、注意ゲーティング（P1/N1, 100–200 ms）が意識アクセス（P3, ~300 ms）に先行するという潜時所見と並行する。認識路が引転を識・速行の前に明示的に置くことは、2500 年前のこの潜時構造の予見である。

### 1.4 神経時間段階との対応

| 認識路 | 皮質段階 | 概略潜時 |
|---|---|---|
| 眼識 | V1 | 50–80 ms |
| 領受 | V2–V4 腹側経路 | 100–150 ms |
| 推度 | IT | 150–200 ms |
| 確定 | LPFC / ACC | 200–300 ms |
| 速行 × 7 | 全脳放送 / 前頭頭頂網 | 300–500 ms |
| 彼所縁 | 海馬・固定化尾部 | 500+ ms |

評価:**中**（IntechOpen 2025 章および deCharms 1997 が明示）。

### 1.5 有分 ↔ DMN / トニック活動

Srinivasan (2020) ほかが有分をデフォルトモードネットワーク（DMN）に対応づけてきた。三点で適合するが、三点で齟齬:(1) DMN は内容豊富、有分は内容極小;(2) DMN は課題陽性網と競合、有分は基底かつ間隙;(3) 有分は業の異熟価値を担う、DMN は倫理中立。より適切な対応は **閾値下のトニック基底状態**。評価:DMN 対応は **弱〜中**、トニック基底対応は **中**。

### 1.6 M1–M4 と脳機能アーキテクチャ

四モジュール分解は感覚運動階層に粗く対応する。**速行のみ業を生む**ことは、随意・報告可能・運動表現可能な処理のみがエージェンシー帰属の座となるという認知神経科学の知見と並行する。評価:**弱**（構造のみ）。

## 2. 説一切有部 *Abhidharmakośa* ↔ 計算論モデル

### 2.1 モデル概要

世親『阿毘達磨倶舎論』とヴァイバーシカの『大毘婆沙論』は、一つの **citta** と最大 46 の **caitta** の **同時相応生起（*saṃprayoga* / 相応）** として認識を分析する。**十大地法（*mahābhūmika*）**（受・想・思・触・作意・欲・勝解・念・定・慧）はすべての心刹那に共起する普遍的心所である。

因果構造:
- **四縁**:因縁・等無間縁・所縁縁・増上縁。
- **六因**:能作因・倶有因・同類因・相応因・遍行因・異熟因。

**三世実有（*sarvāstitva*）**:法は過・現・未に存在、現在性は作用（*kāritra*）の時間。各法は **自性（*svabhāva*）** を持つ。

### 2.2 PDP / コネクショニズムへの対応

**説一切有部が PDP に最もよく対応する**。『大毘婆沙論』の「諸法は単独では弱く、共に在って初めて作用する」（Dhammajoti 2009: 175–176）は、コネクショニズムの「共活性パターンが認知を担う」と **構造的に同一** である。

| 説一切有部の特徴 | PDP / コネクショニズム |
|---|---|
| 心・心所の同時生起 | 特徴ユニットの並列共活性化 |
| 諸法の相応（*saṃprayoga*） | 同期発火 / ヘブ的集合 |
| 十大地法 | 「常時起動」サブネット |
| 自性を持つ性質粒子 | 特性的活性化プロファイルを持つユニット |
| 心と心所の **結合** が内容を生む | 分散表現 |

評価:**中**（構造）、**弱**（公刊文献）。

### 2.3 四縁 ↔ 動力学

| 縁 | 計算論的対応 | 質 |
|---|---|---|
| 因縁 | 因果的事前 / 生成パラメータ | 中 |
| **等無間縁** | **マルコフ的遷移 / 自己回帰更新 / RNN 隠れ状態遷移** | **中〜強** |
| 所縁縁 | 観測 / 尤度 / 入力 | 中 |
| 増上縁 | ゲイン / 精度変調 | 中 |

等無間縁は意識を **同種ストリームにおける一次マルコフ過程** とする ── RNN と自己回帰生成モデルの構造そのものである。

### 2.4 六因 ↔ 型付き因果グラフ

| 因 | 計算論的対応 |
|---|---|
| 能作因 | 背景的・許容的事前 |
| 倶有因 | 双方向 / 循環因果エッジ |
| 同類因 | 同ストリーム内の時間的自己回帰 |
| 相応因 | PDP のコ・アクティベーション制約 |
| 遍行因 | 潜在的混同変数 / 傾向性事前 |
| 異熟因 | **クリーンな対応なし**──倫理的に荷電した長期遅延 |

六因は **豊富に型付けされた因果グラフ** を構成する。Pearl 流の因果ネットが許容する型分化を標準ベイズネットが単一型に縮約するのに対し、六因は明示的に区別する。評価:**Speculative**。

### 2.5 三世実有とブロック宇宙

| 説一切有部テーゼ | 計算論的対応 |
|---|---|
| 法は三世に実在 | 生成モデルの無時間的構造 |
| 作用は現在のみ | 時間索引化された推論 / 方策実行 |
| 自性は三世通じて不変 | パラメータ θ は緩変動/準固定 |

Friston の NESS におけるマルコフブランケット定式化はエージェントを時間不変な統計構造として扱う。**不対応**:FEP は自由エネルギー最小化の流れに、説一切有部は時間三性に業因果を保証させる。機能的に異なる。評価:**Speculative–弱**。

### 2.6 十大地法 ↔ 意識経験の最小条件

十大地法はあらゆる心刹那に普遍する ── **IIT 公理**（内在性・構成・情報・統合・排他性）、**GWT 必要条件**（全脳的利用可能性・点火・報告可能性）、**能動推論最小条件**（事前・尤度・精度・方策・価値を備えた生成モデル）と同型の形式である。

| 大地法 | 計算論的対応 |
|---|---|
| 受 | 価値 / 内受容推論（Seth 2013;Solms & Friston 2018） |
| 想 | カテゴリ的知覚 |
| 思 | 方策選択 / 運動意図 |
| 触 | 感覚接触での予測誤差生成 |
| 作意 | 精度重みづけ / 注意 |
| 欲 | 目標 / 選好事前 |
| 勝解 | 事後確定 / 決定 |
| 念 | 隠れ状態記憶 / 文脈保持 |
| 定 | 注意安定 / 低エントロピー事後 |
| 慧 | 推論的識別 / モデル比較 |

評価:**Speculative–中**。IIT 公理 vs. 大地法の系統的比較は未踏。

## 3. 瑜伽行派（*Vijñānavāda*）↔ 計算論モデル

### 3.1 モデル概要

瑜伽行派（無著、世親『唯識三十頌』『二十論』、安慧、玄奘『成唯識論』）は意識を **八識** に再編する:前五識・第六意識・**第七末那識**（汚染意・アーラヤを自我と執する）・**第八阿頼耶識**（蔵識、種子 *bīja* を保持し経験により薫習 *vāsanā* される）。

各認識事象は **四分** によって分析される:**相分**（対象像）・**見分**（主観像）・**自証分**・**証自証分**。

中心テーゼは **唯識（*vijñaptimātratā*）**:外的対象と内的主観として現れるものは、**意識自身の内側で分岐生成される**。Dan Lusthaus『仏教現象学』(2002) は影響力ある現象学的解釈を示した:唯識は形而上学的（「心のみ存在」）テーゼでなく、現象学的（「経験は完全に意識の提示である」）テーゼである。

認識の方向は **内→外**:阿頼耶識の種子が現象世界を投射し、顕現する経験が種子を薫習する。素朴な順序立った知覚とは逆向きである。

### 3.2 予測処理 / 能動推論への対応 ── 最良の整合

**瑜伽行派が PP / 能動推論に最もよく対応する**。

**(a) 内→外の方向性とトップダウン優位性**。Clark『不確実性のサーフィン』(2016) と Hohwy『予測する心』(2013) は **トップダウン予測がボトムアップ信号を先取りし支配する** ことを明示する。Friston (2010):脳は「世界を反芻する」;『二十論』:世界は唯識、意識自身の構成物以外の外的対象なし。

**(b) 自己証拠化（self-evidencing）と最小自己**。Friston (2018) の「I think therefore I am」「Am I self-conscious?」議論は、**期待自由エネルギーを最小化するあらゆる生物は自身を行為の隠れ原因としてモデル化しなければならない** ── 自己モデリングは自己証拠化から派生する ── ことを論じる。これは末那識が阿頼耶識を「我」と執する瑜伽行派的描写と構造的に同型である。

**(c) 四分と自己モデリング・アーキテクチャ**:

| 四分 | 計算論的対応 | 質 |
|---|---|---|
| 相分 | 生成された感覚予測 | 中 |
| 見分 | 推論された観察者 / 一人称視点 | 中 |
| **自証分** | **現象的自己モデル PSM（Metzinger 2003）/ Friston (2018) 自己証拠化** | **中〜強** |
| **証自証分** | **メタ意識 / 高次表象（Sandved-Smith et al. 2021）** | **中** |

要諦は **主体と客体がともにモデル内で生成される** こと。Metzinger の **透明な自己モデル** ── システムは自己モデルをモデルとして認識できず、モデルは自己として現れる ── と一致する。末那識（我見・我愛・我慢・我癡）は PSM の四属性（一人称視点・所有感・行為主体感・モデル内容との同一化）の正確な機能記述である。Deane (2021) は能動推論枠内で **自我溶解=自己モデル崩壊** とし、瑜伽行派の **転依（*āśraya-parāvṛtti*）** と構造的に並行する。

### 3.3 阿頼耶識 ↔ 階層生成モデル

| 阿頼耶識の特徴 | 生成モデル対応 |
|---|---|
| 種子を蓄える | **学習されたパラメータ / 事前** |
| 種子が顕現経験を生成 | 事前がトップダウン予測を生成 |
| 経験による薫習 | **予測誤差駆動の学習** |
| 意識下作動 | 下位人格的計算 |
| 業の潜在を運ぶ | 長期間クレジット割当 / 超事前 |
| 顕現識と同時 | 生成モデルは常時起動 |

Waldron『仏教的無意識』(2003)、Thompson『目覚め、夢見、存在』(2014)、Srinivasan (2020) が中心的に論じる。評価:**中〜強**。

種子・薫習ループは **予測処理の知覚 / 学習双対性と構造的に同一**:予測が知覚を形成し、予測誤差がパラメータを更新する。

### 3.4 種子論 ↔ 事前 / 重み / 潜在変数

| 種子の特徴 | 計算論的対応 | 質 |
|---|---|---|
| 各種子は潜在的傾向性 | 潜在変数 / 事前 | 中 |
| 種子 → 果 | サンプリング / 事後予測 | 中 |
| 新経験 → 新種子 | オンライン学習 | 中 |
| 世を超えて持続 | 継承事前 / 超事前 | **限界** |
| 倫理的価値を帯びる | **クリーンな対応なし** | 限界 |

### 3.5 唯識 ↔ 暗室問題、インターフェース理論、モデル束縛的心

**(a) FEP の暗室問題**。有機体が自由エネルギー最小化を必須とするなら、なぜ暗く予測可能な部屋に引きこもらないのか。Friston / Clark (2017) の解決:有機体は **多様で生命維持的状態を予測する深い事前** を持つ。瑜伽行派の類似問題:阿頼耶識が自身の世界を投射するなら、なぜ世界は多様で法則的に現れるのか。答えは **業種子（*vipāka-bīja*）** と **共有世界テーゼ**（器世間 *bhājana-loka* は類似業流が共有する）。両者は同型の問題に同型に応える。評価:**中〜Speculative**。

**(b) ホフマンのインターフェース理論**。Hoffman (2015, 2016):知覚は **適応度同調的 UI であって真表象でない**。瑜伽行派と直接的に対応する。評価:**中**。

**(c) PP の「世界に直接アクセス不可」テーゼ**。Clark, Hohwy, Seth:我々は脳の最良推論モデルにのみアクセスする ── 唯識テーゼの現代版。評価:**中**。

### 3.6 阿頼耶識 ↔ 認知的無意識 / 計算精神医学

計算精神医学では、学習された事前とその精度設定が反芻・幻覚・うつ・PTSD を説明する。阿頼耶識の「業的に条件づけられた閾値下傾向が知覚を形成する」役割はこの概念化に対応する。Waldron (2003):阿頼耶識=「仏教的無意識」。評価:**中**。

## 4. 横断的比較分析

### 4.1 最良適合表

| 計算論的枠組 | 最適アビダルマ | 理由 |
|---|---|---|
| **GWT / GNW** | **上座部 citta-vīthi** | 直列・離散・閾値超え点火（速行） |
| **PDP / コネクショニズム** | **説一切有部** | 同時相応、「単独では弱、共には作用」 |
| **予測処理 / 能動推論 / FEP** | **瑜伽行派** | 内→外生成、阿頼耶識=生成モデル |
| **インターフェース理論** | **瑜伽行派** | 非真理的・適応度 / 業同調 UI |
| **現象的自己モデル** | **瑜伽行派（末那+自証分）** | 透明な自己モデルが末那識と対応 |
| **IIT 公理** | **説一切有部 十大地法** | 意識刹那の最小条件 |
| **Posner 注意三網** | **上座部 引転** | 覚醒・定位・実行制御の三層 |

### 4.2 「中央実行者なし」テーゼ

三伝統とも実体的認識者を否定する。これは FEP のホムンクルス不在、GWT（カルテジアン劇場の否定）、PDP、Metzinger の無自我論と **構造的に同一**。評価:**強**。

### 4.3 連続性問題:三解決、三計算論的対応

| 伝統 | 連続性解決 | 計算論的対応 |
|---|---|---|
| **上座部** | **有分** | **トニック神経基底 / 閾値下状態** |
| **説一切有部** | **三世実有** | **無時間的生成モデルパラメータ / NESS のマルコフブランケット** |
| **瑜伽行派** | **阿頼耶識** | **深層生成モデルパラメータ / 学習された超事前** |

本報告で最も情報量のある横断的結果。三伝統は連続性問題に対し、独立に、三つの異なる計算論的解（**状態的・構造的・パラメトリック記憶的**）を発見していた。

### 4.4 注意問題:三伝統横断の精度重みづけ

- **上座部 *manasikāra* / 引転**:Ganeri (2017) が中心に置く;Laukkonen & Slagter (2021) が FA/OM/ND 瞑想における精度変調として運用化。
- **説一切有部 *manaskāra***:十大地法に含まれ、すべての意識刹那で普遍 ── FEP が精度に要請する形式そのもの。
- **瑜伽行派 *manaskāra***:五遍行心所;末那識自体が恒常的バイアス付き精度フィルタとして機能。

Tal et al. (2025) のレビュー:**精度重みづけは瞑想現象学の能動推論モデル全体を貫く共通機構**。アビダルマの作意普遍命題はこれを先取りする。評価:**強**。

### 4.5 刹那 / *citta-khaṇa* 収束

刹那論は三伝統共有（経量部の「映画的」モデルが最も離散フレームに近い）。VanRullen & Koch (2003):知覚は α 帯域振動に紐づく離散フレーム（~10 Hz, ~100 ms）。White (2018):「意識的知覚が離散時間フレームに収まるという思想は…仏教思想に 2000 年以上前に現れた」。ただし仏教の刹那持続（ミリ秒以下）は近代の知覚フレーム（30–400 ms）よりはるかに高速で、亜知覚的粒度を示唆する可能性がある。評価:**中**。

### 4.6 Laukkonen et al. (2025) Contemplative AI 四公理

| 公理 | 最強アビダルマ源流 | 計算論的実装 |
|---|---|---|
| マインドフルネス | 上座部 *sati* + 認識路反省 | 精度の継続的メタ認知監視 |
| 空（*śūnyatā*） | 中観派による三アビダルマ批判 | 硬直事前の弛緩 / 独断的事後の防止 |
| 不二 | 瑜伽行派の能取・所取（*grāhya-grāhaka*）分離批判 | 自己他者モデル境界の溶解 |
| 無辺の慈悲（*bodhicitta*） | 大乗倫理 | 普遍的苦減少を促す選好事前 |

瑜伽行派色の濃い公理（不二、自我事前弛緩としての空）は能動推論アーキテクチャに最も自然に乗る。AILuminate 安全ベンチで d=0.96、囚人のジレンマで d=7+ の報告。評価:**中**。

### 4.7 倫理 / 業次元 ── 厳しい不対応

アビダルマでは **すべての心刹那が倫理的価値**（*kusala / akusala / avyākṛta*）を帯び、業的に有効。計算論的状態は **存在論的に倫理中立**。Contemplative AI (Laukkonen et al. 2025) でさえ倫理を **外的に** 導入する。**第二の限界**:計算論的枠組は倫理を層としてモデル化できても、刹那ごとの内在的倫理荷電を再現できない。

### 4.8 涅槃 / 無為 ── 絶対的限界

無為（*asaṃskṛta*）── 上座部の涅槃、説一切有部の三無為、瑜伽行派の六無為 ── は定義上 **条件によって生成されない**。計算論的枠組は徹頭徹尾条件づけられている（FEP の自由エネルギー最小値、GWT の点火、IIT の Φ 最大、能動推論の選好結果はすべて条件づけ）。**無為に計算論的アナログは存在しない**。最も洗練された Contemplative AI でさえ止滅様の振る舞いをシミュレートできても条件性自体からの脱出はできない。評価:**N/A — カテゴリ的限界**。

### 4.9 メーシーと相互因果

Macy (1991) は **縁起** をシステム理論的相互因果と構造的に同一とする。三伝統に当てはまるが、上座部 *Paṭṭhāna* の二十四縁（*paccaya*）で最も精緻化される。Karunadasa (2010) が最も平易な解説。二十四縁を **豊富に型付けされた因果グラフ** に対応づける形式的研究はまだ存在しない ── **未踏のフロンティア**。

## 結語

三伝統のアビダルマ認識過程モデルは、千年以上にわたる観想哲学的分析を通じて、現代認知科学における **三つの異なる計算論的パラダイム** にそれぞれ対応する。**上座部の *citta-vīthi* はグローバル・ワークスペース理論に**、**説一切有部の阿毘達磨倶舎論は並列分散処理に**、**瑜伽行派の唯識は予測処理 / 能動推論に** 最も近い。三伝統は連続性問題への三つの異なる解（トニック基底・無時間的構造・学習されたパラメータ）を独立に発見し、注意=精度重みづけにおいて収束し、いずれも中央実行者を否定する点で FEP・GWT・PDP・Metzinger の無自我論を予示した。しかし、すべての認知刹那の **業的・倫理的価値** と **無為としての涅槃** には計算論的アナログがない。最も有望なのは、純粋な計算論的還元でも純粋な救済論的拒絶でもなく、Chris Fields の言う **部分的ロゼッタストーン** ──各伝統の過程アーキテクチャを以て計算論的枠組を挑戦し精緻化し、逆に枠組を以てアビダルマの主張を実験的に扱いやすくする──である。無為は、その本性により、すべての地図の縁に留まり続けるだろう。
